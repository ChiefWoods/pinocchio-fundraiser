## Getting Started

### Prerequisites

Update your Solana CLI and rustup.

```bash
agave-install update
rustup default stable
```

### Setup

Clone the repository.

```bash
git clone https://github.com/ChiefWoods/pinocchio-fundraiser.git
```

Build the program.

```bash
cargo build-sbf
```
